# springmvc-basic

Spring MVC 기본 기능

* @RequestMapping 사용 시 지원하는 기능들
* 요청 데이터 전달 방법
  * Query Parameter, HTML Form
  * @RequestParam
  * @ModelAttribute
  * 일반 텍스트
  * JSON
* 응답 데이터 전달 방법
  * Static Resources
  * View Template
  * Message Body (ResponseBody, ...)

관련 Spring MVC 핵심 인터페이스

* HandlerAdapter
* HandlerMapping
* ArgumentResolver
* ReturnValueHandler
* HttpMessageConverter
